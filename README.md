# 개발 프로젝트 (끝말잇기)

> master 브랜치만 이용할 때

## 조장 (GitHub 원격저장소 초대)

* 기본 개발 소스 코드
  * git 저장소 만든후 > github원격저장
    * 초대
    * settings > manage access > invite a collaborator
      * 조원의 username 등록

## 조원

* 이메일로 초대 수락

  ```bash
  $ git clone ____
  # clone은 저장소의 이름의 폴더를 만들어줌
  $ cd 폴더이름
  (master) $
  ```

## 반복

1. 끝말잇기
2. add > commit
3. push
4. 다른조원에게 pull 받으세요

___

pull을 받고, 끝말잇기, add>commit>pull>pull
